# Exploratory Data Analysis of Indian Sectoral Indices

Python-based exploratory data analysis (EDA) project focused on the performance, volatility, correlation, and risk-return characteristics of major Indian sectoral indices between 2021 and 2026.

---

# Project Objective

The objective of this project is to analyze sector-wise market behavior using Python-based financial analytics and data visualization techniques.

Key areas analyzed:
- Sectoral performance trends
- Volatility patterns
- Correlation between sectors
- Rolling risk behavior
- Risk-return tradeoffs

---

# Sectoral Indices Analyzed

- NIFTY BANK
- NIFTY IT
- NIFTY FMCG
- NIFTY PHARMA
- NIFTY AUTO

---

# Tools & Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- yfinance
- Google Colab

---

# Key Analyses Performed

## 1. Comparative Sector Performance Analysis
- Normalized sectoral index comparison
- Relative growth trajectory analysis

## 2. Volatility Analysis
- Annualized sector volatility calculation
- Risk comparison across sectors

## 3. Correlation Analysis
- Correlation heatmap visualization
- Sector relationship analysis

## 4. Rolling Volatility Analysis
- 120-day rolling annualized volatility
- Dynamic sector risk analysis

## 5. Risk vs Return Analysis
- Annualized return vs volatility comparison
- Risk-return tradeoff visualization

---

# Key Findings

- NIFTY AUTO emerged as the strongest performing sector during the analysis period.
- NIFTY FMCG exhibited the lowest volatility and most stable return behavior.
- NIFTY IT showed high volatility with comparatively weaker return efficiency.
- Correlation analysis suggested moderate diversification benefits across sectors.
- Rolling volatility analysis highlighted changing market risk regimes over time.
