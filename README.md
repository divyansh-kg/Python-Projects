## Projects

### 📈 NIFTY 50 Weekly Directional Prediction & Backtesting Model
* **Objective:** Developed an end-to-end machine learning pipeline to predict whether the NIFTY 50 index will close higher or lower week-over-week.
* **Key Achievements:** Optimized an out-of-sample directional prediction accuracy to **59.18%** using an engineered Random Forest classifier. Developed a vectorized compounding backtest demonstrating a **39.42% cumulative return**, successfully outperforming the NIFTY 50 Buy & Hold benchmark (**38.63%**).
* **Core Concepts:** Chronological train/test split (preventing lookahead bias), rolling feature engineering, and variance reduction via strict tree-depth regularization.
